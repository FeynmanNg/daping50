# MyGoCircleMenu 圆形菜单

提示

MyGoCircleMenu 实现了一个圆形菜单，文档MyGoCircleMenu，相关组件 MyGoHtml 文档MyGoHtml

# 基础用法

MyGoCircleMenu 必须与 MyGoHtml 一同使用, 须在 MyGoHtml 定义 menu-name 参数，该参数名称会在模板方法中通过组件的 htmlInfo 对象提供出来
