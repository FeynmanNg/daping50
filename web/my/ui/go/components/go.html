# MyGo 组件

提示

MyGo 为主组件， 内部提供一个div作为canvas容器，组件MyGo

# 基础用法

基础用法需要设置模板方法,包括 node-template 或 node-template-map, link-tempalte 或 linke-template-map, group-tempalte 或 group-template-map, 常用参数还有 width,height(默认 100%), layout(默认布局), config(diagram 配置),参数的默认值参考 MyGo 文档

# 组件事件

是指 xhd-go 通过$emit 方法抛出的时间，其中 on-ready 事件在图渲染完毕后执行一次, on-load-data 会在每次 nodes 和 links 数据发生变化后调用

# diagram 事件

diagram 事件是通过 my-go 的 events 参数对象传入的。示例：选择节点或者点击图形背景
