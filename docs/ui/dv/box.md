# Box 容器

<api-link href="dv/my-dv-box">MyDvBox</api-link>
是一个非常重要的基础组件，大部分组件都继承它。

## 基础用法
:::demo
```html
<template>
  <div style="height:600px">
     <my-dv-page target="parent">
       <my-dv-box width="400px" height="200px" left="100px" top="100px">
          <my-panel title="标题" fit>内容</my-panel>
       </my-dv-box> 
    </my-dv-page>
  </div>
</template>
```
:::
