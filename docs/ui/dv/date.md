# Date 日期时间

:::tip 提示
继承 <api-link href="dv/my-dv-title">MyDvTitle</api-link>、 <api-link href="components/my-Date">MyDate</api-link>
:::

## 基础用法

:::demo
```html
<template>
    <div style="height: 100px">
      <my-dv-page target="parent" fit>
        <my-dv-date></my-dv-date>
      </my-dv-page>
    </div>
</template>
```
:::

## 心跳

:::demo
```html
<template>
    <div style="height: 100px">
      <my-dv-page target="parent" fit>
        <my-dv-date tick></my-dv-date>
      </my-dv-page>
    </div>
</template>
```
:::
